# Brand Context

> Include this file as context whenever generating a page with AI.

---

## Brand Info

- **Company**: PingCAP
- **Products**: TiDB · TiDB Cloud · TiKV
- **Positioning**: Modern database infrastructure for developers and enterprises
- **Core Users**: Backend engineers, database architects, platform engineering teams

## Tone of Voice

- **Professional, direct, developer-focused** — Avoid marketing fluff (e.g. "revolutionary", "game-changing", "cutting-edge")
- **Lead with data** — Use concrete metrics when available (99.99% Availability · Petabyte-Scale)
- **Concise and sharp** — Headlines use verb phrases (Deploy TiDB Your Way · Built for Real-Time Apps)
- **Neutral and honest** — No exaggeration, no disparaging competitors

## CTA Copy Guidelines

| Context | Primary CTA | Secondary CTA |
|---------|-------------|---------------|
| Product pages | Start for Free | View Demo |
| Campaign landing pages | Get Started | Learn More |
| Content pages | Try TiDB Cloud | Read the Docs |

## Forbidden Phrases

- revolutionary / game-changing / industry-leading
- best in class (without supporting data)
- easy to use (always specify *what* makes it easy)
